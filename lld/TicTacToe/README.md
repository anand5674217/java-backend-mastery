# TicTacToe

Low-level design implementation for a Tic-Tac-Toe game using Java 21 and Spring Boot.

## What is included

- Core game models: board, cell, move, player, bot, game state
- Winning strategies: row, column, diagonal
- Maven wrapper and project configuration

## Notes

- The code has been copied as-is from the source project.
- No existing Java code was modified while moving it under `lld`.
- `GameController` is currently a stub, so treat this as an LLD/domain-model exercise rather than a finished API.

## Run

```bash
./mvnw spring-boot:run
```

On Windows, use `mvnw.cmd spring-boot:run`.
